# About App

## Description

This is simple news project to implement full stack with golang

## Technology

1. Golang
2. Fiber
3. Templ
4. Bootstrap
5. AlpineJs
6. Webpack
7. Node

# Requirement

## Without Docker

1. Golang version 1.23.1/latest
2. Node version >= 20
3. Air

## With Docker

1. Node version >= 20
2. Docker

# How to Run

## Without Docker

1. npm i
2. npm run go-webpack
3. air .

## With Docker

1. npm i
2. npm run go-webpack
3. docker compose up
